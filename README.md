# IKEA-E-Commerce-Database-Management-System

Overview

This project focuses on designing a Database Management System (DBMS) for an IKEA-inspired e-commerce platform. The system is intended to manage the core operations of an online furniture store, including customer management, product catalog, inventory, orders, payments, and delivery tracking.

Objective

The main objective of this project is to understand the design and implementation of an e-commerce database by applying DBMS concepts to a real-world business scenario.
